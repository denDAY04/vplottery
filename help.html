FAQ

What are the system requirements?
To install and run the application you must have Microsoft .NET Framework 4.5 and Windows 7 or later.
It is a small, simple application. Why the need to install it?
Yes the application is very small and simply, but installing it makes it easy to associate file extensions with the program. This means .vplf files will both have a custom icon, and double-clicking on them will allow you to open the file with the application directly.
Windows blocks the installer from running?
The source of this is akin to the answer given to the question below this one. If you are running Windows 8 or 10, and the message is given by SmartScreen, you can proceed with the installation by clicking More Info at the end of the message and then choose Run Anyway.
Why does Windows complain about the application being unsafe?
To avoid such warnings require applications to be signed by specific companies, with specific certificates. This costs money, and since this application is only meant for use in the Pearl I have not wanted to spend money on it. If you do not feel safe ignoring the warnings, you are welcome to not use the application.
Where is the application installed?
By default the application is installed in C:\Program Files (x86)\Velvet Pearl\Lottery\ but you may have chosen a different location when installing.
When I try and open the application nothing happens?
This may be your antivirus blocking the program from executing, as per the lack of safe signing as stated above. A way to fix this is whitelisting the program's installation folder with your antivirus. This folder is per default C:\Program Files (x86)\Velvet Pearl\Lottery\. On how to whitelist a file or folder, see the documentation for your antivirus.
How do I remove the program?
You can either remove the program by running the install again and choose 'Remove', or by going to Programs and Features (also known as Add/remove Programs or Uninstall Programs) in Windows Control Panel. From there it can be removed as well. It will also remove the installation folder if no custom files have been placed in it, such as saved lottery files (.vplf files).
How do I contact the application author?
If you have other questions, design suggestions, or found a bug, you can use the contact link in the top of this page to contact me on enjin. Either send me a direct message, a private message, or write on my wall.
How do I see the version of my install?
The version of the installed application can be found by going to Windows Controlpanel, in the Add/Remove Programs section (also known as Uninstall Programs), and then finding Velvet Pearl Lottry. One of the shown column will have a version number.
Does the application automatically update?
Not as of yet. To update to the newest version you have to download the latest installer and run it. The installer will then remove your old version and install the new one. This will not alter any saves you mmight have created.
What does the application save its files as?
The application saves data in a .vplf file format, and will only accept these. It is, however, plain XML and any standard word processor or browser will be able to open these files.
