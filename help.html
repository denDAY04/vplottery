FAQ

What are the system requirements?
To install and run the program you must have Microsoft .NET Framework 4.5 and Windows 7 or later.
It is a small, simple program. Why the need to install it?
Yes the program is very small and simply, but installing it makes it easy to associate file extensions with the program. This means .vplf files will both have a custom icon, and double-clicking on them will allow you to open the file with the program directly.
Windows blocks the installer from running?
The source of this is akin to the answer given to the question below this one. If you are running Windows 8 or 10, and the message is given by SmartScreen, you can proceed with the installation by clicking More Info at the end of the message and then choose Run Anyway.
Why does Windows complain about the program being unsafe?
To avoid such warnings require programs to be signed by specific companies, with specific certificates. This costs money, and since this program is only meant for use in the Pearl I have not wanted to spend money on it. If you do not feel safe ignoring the warnings, you are welcome to not use the program.
Where is the program installed?
By default the program is installed in C:\Program Files (x86)\Velvet Pearl\Lottery\ but you may have chosen a different location when installing.
When I try and open the program nothing happens?
This may be your antivirus blocking the program from executing, as per the lack of safe signing as stated above. A way to fix this is whitelisting the program's installation folder with your antivirus. This folder is per default C:\Program Files (x86)\Velvet Pearl\Lottery\. On how to whitelist a file or folder, see the documentation for your antivirus.
How do I remove the program?
The program can be removed by double-clicking the Uninstall shortcut in the program's install folder. Alternative you can run the installer again and choose 'Remove', or uninstall the program by going to Programs and Features (also known as Add/remove Programs or Uninstall Programs) in Windows Control Panel.
How do I contact the program author?
If you have other questions, design suggestions, or found a bug, you can use the contact link in the top of this page to contact me on enjin. Either send me a direct message, a private message, or write on my wall.
How do I see the version of my install?
The version of the installed program can be found by going to Windows Controlpanel, in the Add/Remove Programs section (also known as Uninstall Programs), and then finding Velvet Pearl Lottry. One of the shown column will have a version number. Alternatively you can navigate to the program's install folder, right-click on Velvet Pearl Lottery.exe and choose properties. Under the Details tab in the top of the window there will be an entry for Product version and File version. Either of these are the version of your installation.
Does the program automatically update?
As of version 1.1.0 the program will now automatically check for updates when it is started, if it has access to the internet and velvetpearl.enjin.com is reachable. If an update is found it will ask you whether you want to download and install the update.
What does the program save its files as?
The program saves data in a .vplf file format, and will only accept these. It is, however, plain XML and any standard word processor or browser will be able to open these files.
